Build a complete, single-file React (.jsx) artifact called "ResumeAI — Context-Aware Resume Analyzer". 
This is a production-grade, fully working AI SaaS app that uses the Anthropic API (Claude) 
to analyze resumes intelligently. Every feature below must be fully implemented and working.

════════════════════════════════════════
TECH CONSTRAINTS (HARD RULES)
════════════════════════════════════════

- Single .jsx React artifact — everything in one file
- Use Recharts for all charts (import from 'recharts')
- Use lucide-react for icons (import from 'lucide-react')
- Use only Tailwind utility classes for styling
- Call Anthropic API at: POST https://api.anthropic.com/v1/messages
  - model: "claude-sonnet-4-20250514"
  - max_tokens: 2000
  - NO Authorization header needed — it is handled by the platform
- Use useState, useEffect, useRef, useCallback hooks
- NO localStorage, NO external auth, NO backend
- File parsing: use FileReader API to read text from .txt files, 
  and show a text extraction notice for PDF/DOCX (ask user to paste text instead, 
  OR use pdf.js from cdnjs for PDF extraction)
- All state lives in React useState
